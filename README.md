# 🌸 Vijaya's Garden at Dusk

My portfolio — a scroll-driven walk through an open flower garden as evening
falls. Six projects bloom along the path, the sky shifts from golden hour to
twilight as you scroll, and the fireflies come out near the end.

**Live:** https://portfolio-zeta-gilt-26xhf2sai6.vercel.app/

### Built with

- Vanilla JS, SVG & Canvas — flowers, foliage, fireflies and petals are all
  generated in code, no image assets
- **GSAP** + **ScrollTrigger** — the pinned bloom animations and the sunset
  timeline
- **Lenis** — smooth scroll
- No build step, no framework — a single `index.html`

### Run locally

Just open `index.html` in a browser. That's it.
